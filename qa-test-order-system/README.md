# 🧪 QA Test — Order System

Este projeto apresenta um teste manual (QA) realizado em um sistema de pedidos, com foco na identificação de bugs e na análise da experiência do usuário.

---

## 🎯 Objetivo

Validar o funcionamento do sistema de pedidos, identificando:

- Bugs funcionais  
- Problemas de usabilidade (UX)  
- Falhas no fluxo de pedidos  

---

## 🔎 Escopo dos Testes

- Criação de pedidos  
- Navegação entre telas  
- Interações do usuário  
- Comportamento do sistema  

---

## 🐞 Bugs Encontrados

- Erros no fluxo de pedidos  
- Comportamentos inesperados durante a navegação  
- Possíveis falhas de validação  

---

## 💡 Análise

O sistema apresenta funcionalidades importantes para o fluxo de pedidos, porém foram identificados pontos que podem impactar a experiência do usuário.
