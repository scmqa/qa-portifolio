# Teste Funcional – Carrinho de Compras (Amazon Brasil)

## Objetivo

Avaliar o funcionamento da funcionalidade de carrinho de compras no site Amazon Brasil, verificando o comportamento do sistema ao adicionar produtos, alterar quantidades, remover itens e atualizar valores exibidos.

---

## Escopo do Teste

O teste foi focado nas seguintes funcionalidades:

- Adição de produtos ao carrinho
- Alteração de quantidade de itens
- Remoção de produtos
- Atualização do valor total
- Persistência dos itens no carrinho durante a navegação

---

## Ambiente de Teste

Sistema testado: Amazon Brasil  
Dispositivo: Desktop / Notebook  
Sistema Operacional: Windows 10 Pro  
Navegador: Google Chrome  
Tipo de teste: Teste manual funcional  

---

## Casos de Teste Executados

| ID | Cenário |
|----|--------|
| TC-CART-001 | Adicionar produto ao carrinho |
| TC-CART-002 | Alterar quantidade de produto |
| TC-CART-003 | Remover produto do carrinho |
| TC-CART-004 | Atualização do valor total |
| TC-CART-005 | Persistência do carrinho |

---

## Resultados

Durante a execução dos testes, as funcionalidades avaliadas apresentaram comportamento consistente, com atualização correta das informações exibidas ao usuário e cálculos de valores realizados sem divergências.

Não foram identificadas falhas nos cenários testados.
