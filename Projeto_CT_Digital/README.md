## Projetos

### 📌 Teste Manual – Carteira de Trabalho Digital

Teste exploratório focado no fluxo de autenticação, incluindo:

- Login via gov.br
- Login via banco (mobile e desktop)
- Verificação em duas etapas (2FA)
- Redirecionamento entre aplicações

*Principais achados:*
- Falhas críticas na geração de código 2FA
- Falha de redirecionamento após autenticação via banco
- Verificação redundante no ambiente desktop

Severidade: Alta  
Impacto: Impede acesso ao serviço.
