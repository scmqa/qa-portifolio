# 🧪 Projeto de Teste Manual - Plataforma Web

## 📌 Sobre o projeto
Este projeto consiste na execução de testes manuais em uma plataforma web, com foco na identificação de falhas, validação de funcionalidades e melhoria da experiência do usuário.

---

## 🔍 Escopo dos testes
- Teste de login
- Cadastro de usuário
- Navegação entre páginas
- Validação de formulários
- Testes exploratórios

---

## 🛠️ Metodologia
- Testes manuais exploratórios
- Execução de cenários reais de uso
- Registro de evidências (prints)
- Documentação em planilha estruturada

---

## 🐞 Resultados
Durante os testes, foram identificados:
- Bugs funcionais
- Problemas de validação de campos
- Possíveis melhorias de usabilidade

---

## 📂 Estrutura do projeto
- relatorio-testes.xlsx → Relatório detalhado dos testes
- relatorio-testes.pdf → Versão em PDF do relatório
- evidencias/ → Prints dos bugs encontrados

---

## 🎯 Objetivo
Demonstrar habilidades em QA manual, incluindo análise de sistema, identificação de bugs e documentação clara e organizada.

---

## 🚀 Tecnologias e ferramentas
- Teste manual
- Excel
- Documentação de bugs

---

## 💡 Observação
Este projeto foi baseado em testes realizados em uma plataforma real, com adaptação de informações para preservar a confidencialidade.
