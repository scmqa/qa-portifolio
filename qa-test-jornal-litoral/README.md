# 🧪 QA Test — Website Jornal A Folha do Litoral

Este projeto apresenta um relatório de testes manuais (QA) realizado no site *Jornal A Folha do Litoral*, com foco na identificação de bugs, análise de usabilidade e melhorias na experiência do usuário.

---

## 🎯 Objetivo

Identificar:
- Bugs funcionais
- Problemas de usabilidade (UX)
- Inconsistências entre dispositivos
- Oportunidades de melhoria

---

## 🔎 Escopo dos Testes

- Navegação geral do site
- Versão Desktop vs Mobile
- Comportamento de rolagem (scroll)
- Exibição de informações (data/hora)
- Organização de conteúdo

---

## 🐞 Bugs Encontrados

### 🔴 Bug 1 — Problema de rolagem no mobile
- A página retorna automaticamente ao topo após um pequeno delay
- Impacta diretamente a navegação e leitura

---

### 🟡 Bug 2 — Inconsistência de data (Desktop vs Mobile)
- A data/hora aparece no desktop, mas não no mobile
- Gera inconsistência na experiência do usuário

---

### 🟢 Bug 3 — Problema de categorização
- Conteúdo “O Agente Secreto” está em “Séries Netflix”
- O correto seria estar na categoria “Cinema”

## 📱 Responsividade

- O site se adapta bem a diferentes dispositivos  
- Porém, apresenta inconsistências entre desktop e mobile  
- A experiência é superior no desktop  

---

## ⚡ Performance

- Possível uso de imagens pesadas  
- Tempo de carregamento pode ser melhorado  
- O delay no mobile impacta a experiência  

---

## 💡 Sugestões de Melhoria

- Melhorar a navegação geral  
- Corrigir comportamento de rolagem no mobile  
- Ajustar categorização de conteúdos  
- Manter categorias atualizadas  
- Otimizar imagens  
- Melhorar consistência entre dispositivos
