# Teste Funcional – Login (Amazon Brasil)

## Objetivo

Avaliar o funcionamento da funcionalidade de login, validando autenticação, tratamento de erros e comportamento do sistema em diferentes cenários.

## Casos de Teste

- Login com dados válidos  
- Senha incorreta  
- Campo obrigatório  
- E-mail não cadastrado  

## Resultado

O sistema apresentou comportamento consistente em todos os cenários testados, com validações adequadas e respostas corretas ao usuário.
